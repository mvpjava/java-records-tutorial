=========================================
Running a .jsh script example from JShell
=========================================

jshell> /open /root/demo.jsh


========================================================
When you want to list the code snippets in jshell script
========================================================
jshell> /list


====================================
Executing individual JShell snippets
====================================
jshell>  /list 1


========================================
Exiting JShell and thus Docker Conatiner
========================================
jshell> /exit
